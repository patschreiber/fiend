## Directory Structure

A core engine feature should have a capitalized name and be a direct child of
the `engine/` directory.

### Good

```plaintext
.src/
├── engine/
│   ├── GameObject
│   │   ├── GameObject.ts
```

### Bad

```plaintext
.src/
├── engine/
│   ├── gameobj/
│   │   ├── GameActor/
│   │   │   ├── GameActorGameObject.ts
│   │   │   └── Player.ts
```
