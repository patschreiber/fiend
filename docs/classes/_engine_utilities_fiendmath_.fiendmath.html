  • clamp(number: number, min: number, max: number): number
    • Defined in engine/utilities/FiendMath.ts:23

    Returns a number whose value is limited to the given range.

    example

let myNum = 10;
FiendMath.clamp(myNum, 0, 255);

    Parameters

    • number: number

      The number to clamp.

      The lower boundary.

    • max: number

      The upper boundary.

    Returns number
