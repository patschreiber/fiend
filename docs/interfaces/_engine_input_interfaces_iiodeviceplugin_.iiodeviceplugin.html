Interface IIODevicePlugin

The IIODevicePlugin interface. Defines an interface for Input/Output devices plugins to guarantee compatibility with the InputHandler.

getInputSignalMap

  • getInputSignalMap(): InputSignalMap
    • Defined in engine/Input/interfaces/IIODevicePlugin.ts:36

    Retrieves the button mapping between the I/O device's input event and the engine's abstract "buttons" that can be assigned to input events.

    example

    For a keyboard I/O device, part of the mapping would look something like:

    {
  "ArrowUp": Button.B1,
  "ArrowDown": Button.B2,
  "ArrowLeft": Button.B3,
  "ArrowRight": Button.B4,
}

    Returns InputSignalMap

    the mapping of input signal from the active I/O device to an internal Button.

    • etc.
